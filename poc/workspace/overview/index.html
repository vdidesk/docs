workspace.accops.cloud

note

Work In Progress

Overview#

Server Address#

Step 1: Please go to https://workspace.accops.cloud

Sign In#

Step 2: Login with Domain Credentials as per below format

Username: firstname.lastname
Password: XXXXXXXXXXXXXXXXXX

2FA Types#

Step 3: After Successful authentication it will prompt for 2FA. Please choose the 2FA Type from Drop Down list.

Below are the 2FA option you will get.
SMS Token : SMS Token will be delivered to Phones as per configured in Active Directory.
Email Token : Email Token will be delivered to Emails as per configured in Active Directory.
Soft/Mobile Token : Accops HyID Mobile client can be installed form Play Store/IOS Store Soft/Mobile Token need to preconfigured before using it as 2FA option, Accops Administrator can Help you out with procedure.
Fido Token : Fido Token Hardware is required to use this option as 2FA.
PC Token : Its a soft Token configured on end User PC.
Hardware Token : Hardware Token
Push Notification : Accops HyID need to installed on End User PC to get the Push Notification PC or Mobile
Biometric Authentication
Face Authentication
  • Go to Accops Workspace Portal (https://sso.accops.cloud)
Here you have to enter the sAMAccountName of the user
For Example:
Username: pravind.kumar
Password: xxxxxx
Doamin: accops.cloud

  • Enter Your Login Credentials, Choose domain from dropdown and click on Sign In.

  • Verify yourself using Multi-Factor Autentcication by choosing the type of MFA from dropdown.

  • Post successful Autentication and Authorization User will will be redirected to Accops Workspace Portal.

Help/Support#

Please send a mail to support@accops.com for further help/Support.
